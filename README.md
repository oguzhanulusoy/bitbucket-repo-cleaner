# About bitbucket-repo-cleaner
This script enables to remove branches which are getting from Bitbucket. You can specify the branches that not to be removed. While running the script, you need to apply each steps respectively.

Prerequisites (You need to fill configuration.yml for Bitbucket API):
- url="...",
- username="...",
- password="..."

Possible actions:
0. Set filename
1. Set project key
2. Set repository slug
3. Get project details
4. Get repository details
5. Get branches
6. Save branches
7. Show not allowed branches
8. Delete branches
(Any to break)
